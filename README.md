# Amazon Top 50 Bestselling Books (2009 - 2019)

Proyecto de análisis de datos realizado en Python.

## Descripción

Análisis de los 50 libros más vendidos en Amazon entre 2009 y 2019. 
El dataset contiene 550 registros categorizados en ficción y no ficción 
según clasificación de Goodreads.

## Dataset

Datos obtenidos de Kaggle:
- Fuente: https://www.kaggle.com/datasets/sootersaalu/amazon-top-50-bestselling-books-2009-2019

### Campos del dataset

| Campo | Descripción | Tipo |
|---|---|---|
| Name | Nombre del libro | String |
| Author | Autor del libro | String |
| User Rating | Calificación de usuarios en Amazon | Number |
| Reviews | Cantidad de reseñas escritas en Amazon | Number |
| Price | Precio del libro (al 13/10/2020) | Number |
| Year | Año en que figuró en el ranking | Number |
| Genre | Género: ficción o no ficción | String |

## Tecnologías utilizadas

- Python
- NumPy
- Pandas
- Seaborn
- Matplotlib
- Jupyter Notebook

## Contenido

- Análisis de los libros más vendidos por año
- Comparación entre géneros: ficción vs no ficción
- Análisis de ratings, reseñas y precios
- Visualizaciones comparativas entre períodos
